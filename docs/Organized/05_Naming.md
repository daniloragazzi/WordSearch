## Modelo de Portfólio

> **1 jogo = 1 app independente na Play Store**

| Aspecto | Regra |
|---------|-------|
| Deploy | Independente por jogo |
| Instalação | Separada para o usuário |
| Package | `com.ragazzistudios.{gamename}` |
| Cross-promotion | Apps promovem uns aos outros |
| Engine | `Core/` compartilhado entre projetos |
